## config zsh for git autocompletion
```cmd
vim ~/.zshrc

autoload -Uz compinit && compinit -u

source ~/.zshrc
```


## config zsh for mysql
```cmd
vim ~/.zshrc

export PATH=${PATH}:/usr/local/mysql/bin/

source ~/.zshrc
```
